# Jailbreak Scripts

## Power Plant Auto-Solver

To run the obfuscated Power Plant auto-solver script in game, execute the following loadstring in your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/huoadf/jailbreak/main/powerplant_solver.lua"))()
```

## How to Uninstall / Deactivate
To take the script down or stop anyone from running it, delete the `powerplant_solver.lua` file from this repository or overwrite its contents with a blank file.
